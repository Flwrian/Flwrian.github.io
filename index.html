Flowgine

Flowgine is a Java chess-engine that uses the Minmax algorithm with alpha-beta pruning. My most time consuming project. It was a great way to learn about optimization and how to make a program as efficient as possible. It was the first time I had to trade off between speed/efficiency and readability.

This project made me use a lot of knowledge and also made me break my head a lot of times but I am proud of the result.

I'm still working on it sometimes to improve it.

  • Java
  • Minimax algorithm
  • Optimization

CompanyPoll

CompanyPoll is a desktop application that allows you to create polls and share them with your employees. It has the top level of security that you can ever imagine. It also uses JavaFX to create a beautiful interface. The app utilizes Zero-knowledge proof for polls to ensure the the poll is not tampered with.

It was a project for my computer science degree.

  • JavaFX
  • Desktop application
